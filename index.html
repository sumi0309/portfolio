Up-To-Date: News Aggregator Web App

A news aggregator application designed to keep users informed about local events and news. [Read More]

This software system, named Up-To-Date, is a news aggregator web application designed to keep users informed about local events and news in their neighborhood. The application caters to various user roles, allowing individuals to sign in, submit news articles with timestamps and locations, search for news using relevant keywords, delete news items, and suggest modifications or contribute additional information.

Designed using MVC architecture in ASP.NET Core, leveraging Razor Pages for dynamic interfaces and Microsoft SQL Server for backend database management, the application ensures efficient handling of user interactions and robust data storage.

The primary goal of Up-To-Date is to provide a platform where users can stay current with the latest happenings in their area, enhancing community engagement and information sharing. The system includes features to ensure ease of use while effectively managing news content and user interactions, making it a valuable tool for local news aggregation and dissemination.

Smart Attendance Portal

A patented software application developed to automate the attendance-taking process for professors. [Read More]

This software was developed to ease the attendance-taking process of professors at our university. The manual process of marking attendance was automated using FaceApi.js, a powerful library for face recognition and detection. Professors could simply upload a group picture of their class onto the application, and the system would automatically recognize each student and mark their attendance.

The system utilized FaceApi.js for backend facial recognition processing and React for a dynamic, user-friendly frontend. This combination enhanced both the efficiency and user experience of the application, offering a seamless interface and robust performance for large classrooms and multiple sessions.

By leveraging advanced technologies and focusing on scalability, this project not only simplified attendance tracking but also demonstrated innovation in applying AI and web development for educational purposes.
